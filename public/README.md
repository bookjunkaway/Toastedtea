# Public assets

- `logo.svg` — Generated approximation of the Book Junk Away logo (yellow dump
  truck, HAULED stamp, palm trees, full tagline). Used as the default brand
  hero image in every template.

## Replacing with your real logo

Drop your own raster file in this folder as **`logo.png`** (1024×1024 PNG with
transparent background recommended). Then in the editor's Brand panel, set the
Hero Image field to `/logo.png` — every template will pick it up automatically.

The SVG version will keep working as a vector fallback at any size, including
4K video export.
